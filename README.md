# 🧩 BO6 Terminus — Calculadora de Easter Egg

Ferramenta web criada para resolver as equações matemáticas do easter egg do mapa **Terminus** de Call of Duty: Black Ops 6.

🔗 **[Acesse online](https://saulomiller.github.io/bo6-terminus)**

---

## 🎮 Como usar

1. Observe os valores de **X**, **Y** e **Z** exibidos no jogo
2. Selecione cada valor correspondente na calculadora
3. Clique em **Calcular**
4. Os três resultados das equações são exibidos na tela

## 🧮 Equações resolvidas

| Equação | Fórmula |
|---------|---------|
| Resultado 1 | `2x + 11` |
| Resultado 2 | `2z + y - 5` |
| Resultado 3 | `y + z - x` |
